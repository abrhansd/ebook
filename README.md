# 🚀 Landing Page - Domine sua Rotina com IA
https://abrhansd.github.io/ebook/
Este repositório contém o código-fonte de uma Landing Page de alta conversão desenvolvida especificamente para a venda do e-book **"Domine-sua-rotina-programe-menos-e-produza-o-dobro-com-IA.pdf.pdf"**. 

O site foi desenhado com uma estética moderna e futurista, utilizando gatilhos mentais e dados extraídos diretamente do conteúdo do guia para maximizar as conversões de venda.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído focando em performance, simplicidade de manutenção e total responsividade:

* **HTML5**: Estruturação semântica da página.
* **Tailwind CSS**: Framework utilitário para um design rápido, moderno e totalmente responsivo (mobile-first).
* **Font Awesome**: Biblioteca de ícones vetoriais para ilustrar as seções de forma intuitiva.
* **Google Fonts**: Tipografia limpa e de alta legibilidade para garantir foco nos textos de venda.

---

## 💎 Funcionalidades e Seções do Site

* **Hero Section de Alto Impacto:** Chamada principal focada na dor do cliente (desperdício de tempo) com um mockup representativo do produto.
* **Prova Social & Estatísticas:** Destaque para dados de mercado (como a economia de até 40% de tempo e triplicação de produtividade com IA).
* **Segmentação de Público:** Divisão clara dos perfis que mais se beneficiam do e-book (autônomos, estudantes, programadores iniciantes e entusiastas).
* **Gatilho de Reciprocidade (Conteúdo):** Resumo modular focado no valor que o leitor vai extrair de cada capítulo.
* **Sessão de Oferta (CTA):** Bloco de fechamento de venda com preço promocional, escassez e selos de segurança/confiança para quebrar objeções.
